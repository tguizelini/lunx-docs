## Rodar a aplicação com o Mock


1. Abra o terminal, navegue até a pasta do projeto e execute o seguinte comando:

```bash
sh start-app.sh
```

## Dados Mock

Na raiz do projeto, existe uma collection do Postman. Basta importar a mesma após finalizar o passo 1 e testar!

## Dados Mock

Os dados mock estão no arquivo "mockData.js". Portanto, caso queira modificar as informações afim de testar outros resultados, é preciso somente alterar esse arquivo.
